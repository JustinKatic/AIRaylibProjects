# AI Pacman Game

DESCRIPTION
variation on pacman game that showcases my AI behaviours

INSTALLATION
visual studios 2019 is required to load files
windows 10

USAGE
To run program open up Release folder and click into exe folder

Controls:

Player movement

W - move forward
S - move backwards
A - rotate tank left
D - rotate tank right


5 - Debug drawing
